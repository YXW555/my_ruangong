# 本地开发环境简单配置

## 配置说明

### 开发环境
- 前端：`npm run serve` → `http://localhost:8080`
- 后端：`docker-compose` → `http://localhost:8081`
- API 请求：直接使用 `http://localhost:8081`

### 生产环境
- 前端：Nginx → `http://服务器IP`
- 后端：Docker → `http://backend:8080`
- API 请求：通过 Nginx 代理 `/api` → `backend:8080`

## 使用方法

### 本地开发

1. **启动后端服务**：
   ```bash
   cd campus-trading-platform
   docker-compose up -d
   ```

2. **启动前端开发服务器**：
   ```bash
   cd campus-trading-platform/frontend
   npm run serve
   ```

3. **访问**：http://localhost:8080

### 生产环境

无需修改，继续使用服务器上的部署。

## 配置原理

- **开发环境**：`baseURL = 'http://localhost:8081'`
  - 前端直接请求后端，不需要代理
  
- **生产环境**：`baseURL = '/api'`
  - 通过 Nginx 代理到后端

## 验证

### 本地开发环境

1. 打开浏览器开发者工具（F12）
2. 查看 Network 标签
3. API 请求应该是：`http://localhost:8081/xxx`
4. 应该能正常返回数据

### 生产环境

1. 访问服务器 IP
2. API 请求应该是：`http://服务器IP/api/xxx`
3. 通过 Nginx 代理到后端
